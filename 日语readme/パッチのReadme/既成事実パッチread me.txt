eratohoK（ver.1.21時点）でオミットされた既成事実関連の表示を行うパッチです。

某所のスレで、「思ったけど霊夢のセリフっておかしくない？恋慕獲得しているのに
会話内容が煮え切らない感じがするのってデフォなの？」という誤爆を見かけて、
「せっかく口上書いてくれてるんなら、ちゃんと表示させてあげたいな」と思い立ち
いっそ全て修正してみようとこのパッチを作りました。
といっても、難しいことは何もしていません（そもそも出来ません）が……。

--------------------------------------------------
※　具体的な修正内容　※

各キャラKOJO_A内の「MARK:2 == ○」を「TALENT:○○」に置換しただけです。
その他は一切手を加えていません

MARK:2 == 0　　→　　TALENT:キス未経験
MARK:2 == 1　　→　　!TALENT:キス未経験
MARK:2 == 2　　→　　TALENT:恋人
MARK:2 >= 2　　→　　TALENT:恋人 || TALENT:合意
MARK:2 == 3　　→　　TALENT:合意

のように変更しています。

また、デフォルトでキス経験済みキャラの初キス判定用に「CFLAG:250」を各分岐に追加、使用しています。
現状、口上フォルダ内にKOJO_Aがあるキャラのみに修正を行っております。

----------------------------------------------------------------
★導入方法
eratohoKのERBフォルダの中に、この「ERB」フォルダを入れて上書きしてください
