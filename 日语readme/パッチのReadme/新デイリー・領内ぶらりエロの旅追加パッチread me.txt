★このファイルはeratohoK用デイリーイベント追加パッチです(動作確認ver.1.22.6)
--------------------------------------------------------------------------------------
■概要■
新規デイリーイベント「領内ぶらりエロの旅」を追加するパッチです。

■具体的な内容■
--------------------------------------------------------------------------------------
拠点コマンド「領地探索」で起こるエロイベントの内、「連絡先を渡された」などで終わっていたイベントの続きが欲しかったので
自分で作ったものです。

各地に赴くことでイベントが起きて、各種エロイベントが続いていきます。
初回イベントは、おおむね元のイベントに沿った物になっていますが、2回目以降は自己解釈で続きを作っています。
たまに同勢力内のキャラと鉢合わせて適当に遊ぶイベントになることがあります。

-----------------------------------------------------------------
■使い方■
-----------------------------------------------------------------
同根のERBフォルダをeratohoK ver.1.22本体が入ったフォルダに入れて、デフォのERBフォルダにそのまま上書きしてください。

既に自前、もしくは他のデイリーイベントパッチ等を導入されている方は
「CREATE_DAILY_MAP」と「DAILY_CONST」の競合にご注意ください。
