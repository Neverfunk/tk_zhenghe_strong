＜内容＞

1.一刀・華琳の嗜好設定の追加

　『設定』の『特殊設定』に『一刀・華琳の嗜好設定』が追加されます

　これは一刀・華琳の餌食対象を設定するもので、『通常』『あなたと面識のある者のみ』『恋慕・服従の者のみ』の三つから選べます


　　1.通常

　　　通常の設定です

　　　特に制限はありません



　　2.あなたと面識のある者のみ

　　　あなたが一度でも会った事のあるキャラ（『会』が白くなっているキャラ）が対象になります

　　　それ以外のキャラは対象になりません



　　3.恋慕・服従の者のみ

　　　恋慕・服従が付いているキャラのみ対象になります

　　　この設定を有効にするためには『親密な女性士官を放置したとき』を『一刀や華琳になびく』に変更する必要があります





2.設定に関する変数の使用

　このパッチでは未使用の変数『CONFIG:82』を新たに使用しています
