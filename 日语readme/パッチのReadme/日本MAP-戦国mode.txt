日本MAP-戦国mode

これはeratohoK 1.6.3版時点で使われることを想定したMAPです
主にEUマップを参考にして製作しました。

WW2があるなら戦国もという発想で作りました
城の名前が違ったり四国の位置や色々な城の位置がズレてたりするのは仕様です

経済力とシナリオは黒田如水氏のものをそのまま入れてあります
防衛に関しては適当に入力しました

何か不具合があれば書き込んでもらえれば幸いです(ただシナリオに関しては対応できないと思います)

・城に関して
できるだけ有名な城を使おうとはしましたが、同じ文字になってしまうので断念･･･
一応備考だけおいておこうかと思います

・実装を諦めた城
盛岡城
 築城が1598年のため
 遅いとはいえないが、やはり早めの城が良かったから

白石城
 名前が白地城と被るため
 続日本100名城の5番手に選ばれた城

岡崎城
 岡豊城と被るため
 松平家にはお引越ししてもらいました

小高城
 名前が小田原城と被るため
 相馬家ならここだけどこれまた断念
 余波として一条家も消滅

小谷城
 名前が小田原城と被るため
 浅井家といえばここなんだけど、致し方ない

信貴山城
 あまりにも近すぎるため
 何か爆弾でも仕込みたかった･･･

大和郡山城
 吉田の方が有名そうだったから
 特に後悔はしていない

二条城
 スペース上の問題、あと将軍山城の方が被らないと思ったから
 少なくとも4回は焼かれた城、二条御所とも言う

黒瀬城
 黒滝城と被るため
 向こうの方は城が少ないと見て変更した城
 黒瀬と松葉、どちらにしても西園寺家の本拠であることに変わりない

中村御所
 中村城と被るため
 実装予定から実装不能へと逆転された城
 小という文字が多いのが悪い･･･

立花山城
岩尾城

勢福寺城

・名前を変えている城
賀田城
 元は大浦城、大友館と被るため変更
 謀反して無事生き残った津軽家の居城

柏葉城
 元は七戸城、七尾城と被るため変更
 七戸家の城

留ヶ崎城
 元は三戸城、三隅城と被るため変更
 南部家の本城といえる

斯波館
 元は高水寺城、高山城と被るため変更
 でも実際は郡山城というらしい、斯波家の城

甲尾城
 元は国府尾城、国府台城と被るため変更
 離島の城はあまり分からないのだ･･･

会津若松城
 元は黒川城、黒滝城と被るため変更
 あまりいい城が越後に見当たらなかったのでこちらを変更
 有名さとか歴史的にとかいう知識などは持ち合わせてないのだ

霧ヶ城
 元は岩村城、岩尾城と被るため変更
 結構武田と織田の争いがあったらしい

兜山城
 元は高遠城、高山城と被るため変更
 色々歴史のある城で、日本100名城にも選ばれている

萱切城
 元は城井谷城、なんとなく被りそうだったため変更
 実際は被らなかったので杞憂、まぁ違うほうが面白いかと思って修正せず(面倒だったわけではない)

多気城
 元は霧山城だったり霧山御所だったり、霧ヶ城と被るため変更
 北畠家の本拠地にして難攻不落の城
 しかし、織田家に目をつけられたのが運の尽きで落城した模様
