★このファイルはeratohoK用の処女関連の口上を表示させるパッチです。
(動作確認ver.1.23) 作：249スレ目>>381
--------------------------------------------------------------------------------------

以前自分が作成した「既成事実修正パッチ」と「処女関連口上表示パッチ」で内容が競合している部分が有ったため、それらをまとめて修正したパッチです。

以前の「処女関連口上表示パッチ」での既成事実部分が、「既成事実修正パッチ」を上書きしてしまっていたので、それを修正しています。
また、各種処女関連の台詞がある口上の性交コマンド時の分岐条件を「IF TCVAR:13 == 1」から「IF EXP:TARGET:Ｖ性交経験 == 1」に変更しています。



----------------------------------------------------------------------------------------------------
★導入方法
eratohoK ver.1.23のフォルダの中に解凍した「ERB」フォルダを入れて上書きする。
