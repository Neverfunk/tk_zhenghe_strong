★このファイルはeratohoK用デイリーイベントの改変パッチです(動作確認ver.1.23)
--------------------------------------------------------------------------------------
■概要■
デフォルトのデイリーイベント「淫紋」の派生デイリーに、新ルートを追加するパッチです。

従来の内容はそのままに、新たに「好感度等が高いと、悪魔ではなくあなたに堕ちる」ルートを追加した物です。

■具体的な内容■
--------------------------------------------------------------------------------------

◆「派生デイリーイベント「LEWDNESS_CURSE_DERIVATION」◆
名前などは元のまま。淫紋を刻まれたキャラが主人公でなく、好感度などの合計値が1万以上の場合に派生します。

基本は従来どおりで、条件を満たしていると男たちに犯された後に、満足できなかったのであなたの元に来る、という流れです。
その際の選択肢で「断る」を選ぶと追加ルートは終了し、以降は従来どおりの展開になります。
（また、対象キャラが霊夢などの特定のキャラで欲望が2以下の場合、１／２の確率で自力で解呪するイベントもあります）

最後までいくと淫紋がつき淫乱になりますが、従来ルートのように寝取られるのではないので恋慕等は消えません。
精液便女も取得しませんが貞操は消えます。

淫紋定着後、男たちに犯されても満足できず、%ANAME(MASTER)%に犯されに来る、という流れになっています。

★最初のイベントに、純愛ルートの選択肢を追加★
最初のイベントで「彼女のことを信じる」を選ぶとフラグが立ち、以降好感度などの合計値が1万以上の場合だと他の男に犯されない別ルートへ分岐します。

-----------------------------------------------------------------
同根のERBフォルダをeratohoK ver.1.23本体が入ったフォルダに入れて、デフォのERBフォルダにそのまま上書きしてください。

既に自前、もしくは他のデイリーイベントパッチ等を導入されている方は
「DAILY_CONST」の競合にご注意ください。
