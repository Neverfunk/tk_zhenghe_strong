★このファイルはeratohoK用デイリーイベントの改変パッチです(動作確認ver.1.27)
--------------------------------------------------------------------------------------
■概要■
デフォルトのデイリーイベント「少女兵士」デイリーで少女を直属の部下にした場合に、対応した種族タグを追加するパッチです。
それとついでに加入時の能力値にランダムで触れ幅を持たせるようにしました。



-----------------------------------------------------------------

-----------------------------------------------------------------
同根のERBフォルダをeratohoK ver.1.27本体が入ったフォルダに入れて、デフォのERBフォルダにそのまま上書きしてください。
